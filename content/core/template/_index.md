テンプレートコードです。

## 説明

空ソースファイルを作成したら、とりあえずこのテンプレートを貼り付けます。

## 副作用
- 標準ライブラリを一括でインクルードするため、 `bits/stdc++.h` をインクルードしています
- いちいち `std` を書くのは面倒なので575しています
- `pair` 型と `vector` 型の値をストリーム出力できるようにしています
  - `pair` は `(x, y)` のような文字列で出力されます
  - `vector` は `[a, b, c]` のような文字列で出力されます

## API

### ll
`long long` 型（64bit整数）の短縮名です。

### vi
`vector<ll>` 型（64bit整数列）の短縮名です。

### ii
`pair<ll, ll>` 型（64bit整数のペア）の短縮名です。

### vvi
`vector<vi>` 型（64bit整数列の列）の短縮名です。

### vii
`vector<ii>` 型（64bit整数のペアの列）の短縮名です。

### vs
`vector<string>` 型（文字列の列）の短縮名です。

### REP
0-indexedのforループのマクロです。

#### 引数
- $i$: ループ変数名
- $n$: ループ回数

#### 置換後のコード
`for (ll i = 0; i < n; ++i)`

### REP1
0-indexedのforループのマクロです。

#### 引数
- $i$: ループ変数名
- $n$: ループ回数

#### 置換後のコード
`for (ll i = 1; i <= n; ++i)`

### FOR
for eachループのマクロです。

#### 引数
- $i$: mutableな参照イテレータの名前
- $a$: ループ対象のコンテナ

#### 置換後のコード
`for (auto &i : a)`

### IN
単一値を標準入力するマクロです。

#### 引数
- $T$: 標準入力する値の型
- $x$: 標準入力する値の変数名

#### 置換後のコード
```cpp
T x;
cin >> x;
```

### AIN
列を標準入力するマクロです。

#### 引数
- $T$: 標準入力する列の各値の型
- $a$: 標準入力する列の変数名
- $n$: 列の要素数

#### 置換後のコード
```cpp
vector<T> a(n);
FOR(i, a) cin >> i;
```

### A2IN
2列を標準入力するマクロです。

#### 引数
- $T1$: 標準入力する列1の各値の型
- $a$: 標準入力する列1の変数名
- $T2$: 標準入力する列2の各値の型
- $b$: 標準入力する列2の変数名
- $n$: 列の要素数

#### 置換後のコード
```cpp
vector<T1> a(n);
vector<T2> b(n);
REP(i, n) cin >> a[i] >> b[i];
```

### OUT
値を標準出力するマクロです。

#### 引数
$x$: 標準出力する値

#### 置換後のコード
`cout << x << endl;`

### FOUT
小数部を必ず15桁標準出力するマクロです。

#### 引数
$x$: 標準出力する値

#### 置換後のコード
`cout << fixed << setprecision(15) << x << endl;`

### JOUT
イテレータを指定した区切り文字で結合した文字列をストリームに出力する関数です。

#### 引数
- $s$: イテレータの先頭
- $e$: イテレータの末尾
- $os$: 出力ストリーム
  - デフォルト値: 標準出力
- $sep$: 区切り文字
  - デフォルト値: 半角スペース

#### 戻り値
$[s, e)$ 内の各要素を $sep$ で区切り文字文字として $os$ に出力した後の $os$

### YES
真偽値を応じた文字列をストリームに出力する関数です。

#### 引数
- $cond$: 出力する文字列を決定するのに用いられる真偽値
- $yes$: $cond$ が真のときに出力する文字列
  - デフォルト値: `Yes`
- $no$: $cond$ が偽のときに出力する文字列
  - デフォルト値: `No`
- $os$: 出力ストリーム
  - デフォルト値: 標準出力

#### 戻り値
$yes$ または $no$ を $os$ に出力した後の $os$

#### 置換後のコード
`cout << fixed << setprecision(15) << x << endl;`

### ALL
コレクションの先頭要素と末尾要素へのイテレータを取得するマクロです。

#### 引数
$a$: 先頭要素と末尾要素へのイテレータを取得するコレクション

#### 置換後のコード
`a.begin(), a.end()`

### SORT
`vector` を昇順ソートするマクロです。

#### 引数
$a$: 昇順ソートする `vector`

#### 置換後のコード
`sort(ALL(a))`

### RSORT
`vector` を降順ソートするマクロです。

#### 引数
$a$: 降順ソートする `vector`

#### 置換後のコード
`sort(ALL(a)); reverse(ALL(a))`

### DUMP
指定した変数の識別子と値を標準出力するマクロです。
デバッグで有用です。

#### 引数
$x$: 出力対象の変数

#### 置換後のコード
`cout << xの識別子 << " = " << (x) << endl;`

### DUMPA
指定したコレクションの識別子と各要素を標準出力するマクロです。
デバッグで有用です。

#### 引数
$a$: 出力対象のコレクション

#### 置換後のコード
`cout << aの識別子 << " = {"; JOUT(ALL(a), cout, ", ") << "}" << endl;`

### INF
ll型の $10^{18}$ です。
64bit整数において、十分に大きい値として使います。例えば番兵として有用です

### MOD
ll型の $10^9 + 7$ です.
数え上げ等で剰余をとる問題において頻繁に使われる値です
